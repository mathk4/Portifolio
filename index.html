Projetos

Monitoramento de Posição no Google Maps

Aplicação desenvolvida em Python para monitorar a posição de comércios no Google Maps com base em termos de buscas específicos.

  • Cadastro de comércios em banco de dados PostgreSQL
  • Busca automática da posição do comércio no Google Maps (via Selenium)
  • Armazenamento do histórico de buscas e rankings no banco de dados
  • Reutilização da última lista de termos de buscas pesquisados
  • Exportação dos resultados para Excel, organizados por data (via Pandas)
  • Validação de endereço automaticamente via API ViaCEP

O projeto foi desenvolvido para apoiar a WA Digital Premium no acompanhamento da visibilidade de negócios locais.

Comunidade NBL - Rede Social Web

Aplicação web desenvolvida com Python e Flask como projeto prático para consolidação de conhecimentos em desenvolvimento Back-end.

O projeto consiste em uma rede social adaptada para um grupo de jogadores de Brawl Stars (NBL), baseada em um modelo proposto durante curso da Hashtag Treinamentos.

Funcionalidades:

  • Sistema de rotas e navegação com Flask
  • Criação e processamento de formulários
  • Integração com banco de dados SQLite
  • Estrutura básica de autenticação e interação entre usuários
  • Interface web responsiva utilizando Bootstrap

Tecnologias utilizadas:

  • Python
  • Flask
  • HTML
  • CSS
  • Bootstrap
  • SQLite

O projeto teve como principal objetivo consolidar fundamentos de desenvolvimento Back-end, incluindo lógica de aplicação, estruturação de rotas e integração com banco de dados, além de proporcionar uma visão prática da comunicação entre frontend e backend.
